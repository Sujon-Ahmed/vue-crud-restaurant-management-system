# restaurant-management-system

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Install JSON Server
```
npm install -g json-server
```

### Start JSON Server

```
json-server --watch db.json
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
